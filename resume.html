Projects

Bike-share analysis: A group project comparing the bike-share programs in New York City and San Francisco, to identify a typical rider profile and trends in each of the cities. Analysis was completed in Jupyter Notebook using Pandas and several libraries including Numpy, Matplotlib, Seaborn and Plotly.
Denver Flight Data: A partner project utilizing numerous datasets from OpenFlights.org that we creatively merged together into one dataset allowing us to create a map of airline routes out of Denver, categorized by airline. The other piece of this project included web scraping of SkyScanner.com to pull current flight deals available. The data was pulled live into a filterable table allowing the user to select a departing airport, month of travel and destination country. Analysis was completed in Jupyter Notebook using Pandas and the data was exported as a geoJSON file to be used in Leaflet.js for the mapping. The web scraping utilized an API and BeautifulSoup then filtered through a dynamic javascript table. Both pieces were presented together through a simple Flask-initiated website.
Twitter Sentiment Analysis: A sentiment anylsis of tweets from five popular news sources, using Twitter Tweepy API, VADER sentiment analysis and Matplotlib
